Detalhes do Projeto

Automação Inteligente de Laudos com IA

Informações do Projeto

  • Categoria: Sistema Especialista / Aplicação de Dados com IA
  • Cliente: UNAX Group
  • Data do Projeto: 2025
  • Tecnologias: Python, Streamlit, Gemini API, Pandas, NumPy, SQL Server, MySQL

Descrição Detalhada

Este sistema especialista resolve o desafio de gerar laudos técnicos complexos para análise de fluidos em plataformas offshore, substituindo um processo manual e demorado por uma solução automatizada, inteligente e precisa.

Arquitetura e Fluxo de Trabalho

  1. Sincronização de Dados (ETL): Um script dedicado (sync.py) conecta-se a um banco de dados SQL Server externo, extrai os dados brutos das análises e os carrega em um banco de dados MySQL local, que serve como fonte principal para a aplicação.
  2. Processamento e Análise: O núcleo do sistema (backend.py) utiliza as bibliotecas Pandas e NumPy para limpar, processar e realizar cálculos técnicos sobre os dados sincronizados.
  3. Motor de IA com Gemini: O diferencial do projeto é a integração com a API do Gemini. Os dados processados são enviados à IA, que, com base em seu vasto conhecimento e no contexto fornecido (potencialmente a partir de uma base de conhecimento em MySQL), gera um diagnóstico técnico detalhado, com a nuance e a qualidade de um especialista humano.
  4. Interface Web Interativa: Uma aplicação web construída com Streamlit (app.py) serve como a interface principal, permitindo que os usuários iniciem análises, visualizem os laudos gerados pela IA e gerenciem o sistema.

Principais Funcionalidades Implementadas

  • ETL Automatizado: Processo robusto para sincronização de dados entre diferentes sistemas de banco de dados.
  • Geração de Diagnóstico por IA: Laudos e recomendações criados dinamicamente pela API do Gemini, garantindo alta qualidade e consistência.
  • Análise e Processamento de Dados: Limpeza e manipulação de grandes volumes de dados técnicos com Pandas.
  • Dashboard Interativo: Interface amigável com Streamlit para visualização de resultados e interação com o sistema.
  • Gestão de Usuários: Script dedicado para o gerenciamento de acesso à plataforma (add_user.py).
